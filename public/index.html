What is this?

This is Synesthesia.

This project was originally created during Spotify DevX Gothenburg 2018 by Mateo Raspudic, Lovisa Hagström, Gök Berk, and Sharan Sabi.
The version of the project that you are viewing now is improved and further developed by Mateo.

This project is based on Color-Wander by Matt DesLauriers but has been adapted and modified to be used with Spotify. Synesthesia gets album images through the Spotify API and then uses Imagga to extract the most prominent colors from the image and uses these to paint the album art beautifully with particles.
